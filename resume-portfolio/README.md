# Resume Portfolio

A simple GitHub Pages-ready resume website for Mallugari Dinesh Naidu.

## Files

- `index.html` — main resume page
- `styles.css` — responsive styling

## Deploy to GitHub Pages

1. Create a new GitHub repository, for example `resume-portfolio`.
2. Commit and push the `index.html` and `styles.css` files to the repository.
3. In the repository settings, enable GitHub Pages and set the source to the `main` branch and root folder.
4. Your resume will be available at `https://<your-github-username>.github.io/<repository-name>/`.

## Notes

- You can rename `index.html` to `resume.html` if you prefer, but GitHub Pages requires `index.html` in the root folder for the default site.
- Update the contact details and project information directly in `index.html` as needed.
